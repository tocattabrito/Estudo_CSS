Bem-vindo!

No CSS nós possuímos os seletores por ID

  • Seletor por Tipo:Busca elementos por uma tag HTML especifica
  • Seletor por ID (#)Busca elementos através do atributo "id"
  • Seletor por Classe (.)Busca elementos através do atributo "Classe"
  • Seletor Universal (*)Seleciona todos os elementos HTML
  • Seletor de atributo ([atrib])Seleciona elementos que possuem um atributo especifico
    em nosso documento HTML e conseguimos também buscar atributos com um valor especifico
